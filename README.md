# MOIM
<br>

> 챌린지를 같이 할 사람을 모으고, 챌린지에 참여하는 건강관리 모바일 웹 서비스

<br>

### Features
* 챌린지를 만들어 함께할 사람 모으기
* 내가 관심 있는 챌린지를 검색하고, 저장하고, 참여하기
* 사진, 태그와 함께 피드를 작성해 일상 공유하기
* 달라진 내 몸무게를 기록하여 그래프로 한 눈에 확인하기
* 내 건강 관리에 도움을 주는 건강 점수와 코멘트
